# 完全免费上线指南（全程不绑卡）

最终方案（三处都免费、都不要信用卡）：
- **网站托管**：Render 免费版（web service）
- **数据库**：MongoDB Atlas 免费版 M0（订单/用户/设置/收款码都存这里，重启不丢）
- **代码仓库**：GitHub（Render 从这里拉取部署）

> 程序已改成"双模式"：设置了 `MONGODB_URI` 就用云数据库，没设置就用本地文件。云上自动走数据库，数据持久。
> 上传的收款码/凭证在云模式下以 base64 存进数据库，跟着持久化。

## 你需要做的（只有"注册账号"这步必须你亲自做，每步我都给精确值）

### 第 1 步：建 GitHub 仓库（放代码）
1. 注册/登录 https://github.com
2. 右上角 + → New repository，名字填 `ai-recharge`，选 **Public**，**不要**勾任何初始化项，Create。
3. 把仓库地址（形如 `https://github.com/你的名字/ai-recharge.git`）发我；
   若愿意让我帮你推送，再在 GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens 生成一个**对该仓库有 Contents 读写权限**的 token 发我，我来 `git push`。
   （不想给 token 也行，我把推送命令给你，你自己粘进 PowerShell 跑。）

### 第 2 步：建免费数据库 MongoDB Atlas（不绑卡）
1. 注册 https://www.mongodb.com/cloud/atlas/register
2. 创建集群时选 **M0 Free**（免费、地区选 AWS / Singapore 或离你近的）。
3. 左侧 **Database Access** → Add New Database User：设一个用户名+密码（**记下来**，密码别用特殊符号以免转义）。
4. 左侧 **Network Access** → Add IP Address → 选 **Allow access from anywhere（0.0.0.0/0）**（Render 的出口 IP 不固定，必须放开）。
5. 回到 **Database** → Connect → Drivers，复制连接串，形如：
   `mongodb+srv://用户名:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority`
   把 `<password>` 换成你的真实密码，并在 `.net/` 后面加上库名 `recharge`，最终形如：
   `mongodb+srv://用户名:你的密码@cluster0.xxxxx.mongodb.net/recharge?retryWrites=true&w=majority`
   **这串发我**（或自己保管，第 3 步要填进 Render）。

### 第 3 步：在 Render 部署（不绑卡）
1. 注册 https://render.com（可直接用 GitHub 账号登录，授权访问你的仓库）。
2. 顶部 **New +** → **Blueprint** → 选你的 `ai-recharge` 仓库 → Apply。
   （仓库里已有 `render.yaml`，Render 会自动按它创建免费 web 服务。）
3. 部署过程中（或在服务的 **Environment** 里）填入环境变量：
   - `MONGODB_URI` = 第 2 步那串连接字符串
   - `AUTH_SECRET` 会自动生成，无需填
4. 等待构建完成（约 2-4 分钟），得到网址：`https://gpt-recharge-xxxx.onrender.com`
   —— 电脑和手机浏览器都能直接打开！

> ⚠️ 上线后第一件事：打开 `网址/admin/`，用 admin/admin123 登录 →
> ①站点设置里关闭"模拟支付"、上传你的支付宝/微信收款码；②尽快改掉默认密码。

> 提示：Render 免费版在无人访问约 15 分钟后会休眠，下次访问需冷启动约 30-60 秒（数据不会丢）。介意的话以后可升级或换付费 VPS。

## 让用户「搜得到」（有了网址后）
1. **Google Search Console**（https://search.google.com/search-console）添加你的网址，提交 `网址/sitemap.xml`。
2. **Bing Webmaster Tools**（https://www.bing.com/webmasters）同样提交。
3. 程序已内置 `robots.txt`、`sitemap.xml`（自动用真实域名）、SEO 标题/描述/关键词、手机端自适应。
4. 收录一般几天~几周；多在社群/公众号/小红书放链接，外链越多收录越快。
