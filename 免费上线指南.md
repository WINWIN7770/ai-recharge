## 推荐方案：Fly.io（真正免费 + 数据持久，最适合本程序）

本程序把订单/用户/设置/收款码存在 `data/` 目录，**必须用带持久卷的平台**，否则重启丢数据。Fly.io 免费额度自带小容量卷，最合适（项目里已备好 `Dockerfile` 和 `fly.toml`）。

### 步骤（全程命令照抄）
```bash
# 1. 安装 flyctl（Windows PowerShell 里运行）
#    iwr https://fly.io/install.ps1 -useb | iex

# 2. 注册并登录（会让你绑一张卡做身份验证，免费额度内不扣费）
fly auth signup      # 或已有账号： fly auth login

# 3. 在项目目录初始化（D:\代充网站 下运行）
#    把 fly.toml 里的 app 名改成全局唯一，例如 aichong-8x2k
fly launch --no-deploy --copy-config --name 你的唯一应用名 --region hkg

# 4. 创建持久数据卷（1GB，免费额度内）
fly volumes create recharge_data --region hkg --size 1

# 5. 设置登录密钥（务必改成长随机串）
fly secrets set AUTH_SECRET=把这里换成一长串随机字符

# 6. 部署！
fly deploy
```
完成后会得到一个网址：`https://你的唯一应用名.fly.dev` —— 电脑和手机浏览器都能直接打开使用。

> 上线后记得登录管理后台（`/admin/`，admin/admin123）→ 站点设置：**关闭"模拟支付"、上传你的收款码、改默认密码**。

## 备选：Railway（点点鼠标更简单，有免费试用额度）
1. 把代码传到 GitHub（见下方"把代码传到 GitHub"）。
2. 打开 https://railway.app → New Project → Deploy from GitHub → 选你的仓库。
3. Railway 自动识别 Dockerfile 部署。
4. 在项目 **Variables** 加：`AUTH_SECRET`=长随机串、`NODE_ENV`=production、`COOKIE_SECURE`=1。
5. 加一个 **Volume**，Mount 到 `/app/data`（否则数据会丢）。
6. Settings → Networking → Generate Domain，得到 `xxx.up.railway.app`。
> Railway 免费是试用额度（约 $5），用完需充值；Fly.io 更适合长期免费。

## 绑定品牌域名（强烈建议，让网址好记好搜）
免费二级域名（`xxx.fly.dev`）能用，但不好记也不利于排名。买个便宜域名（¥30~90/年）后：
- **Fly.io**：`fly certs add 你的域名.com`，再按提示去域名商加一条 CNAME/A 记录。
- **Railway**：Settings → Custom Domain 填域名，按提示加 CNAME。
解析生效后即可用自己的品牌网址访问，并自动配 HTTPS。

## 让用户「搜得到」
有了正式域名 + HTTPS 后：
1. **Google Search Console**（https://search.google.com/search-console）添加域名，提交 `https://你的域名/sitemap.xml`。
2. **Bing Webmaster**（https://www.bing.com/webmasters）同样提交。
3. 程序已内置 `robots.txt`、`sitemap.xml`（自动用真实域名）、首页 SEO 标题/描述/关键词、手机端自适应。
4. 收录一般要几天~几周；多在社群/公众号/小红书放链接，外链越多越快。
> 百度对未备案海外站收录有限，面向国内主要靠 Google/Bing + 口碑分享导流。
